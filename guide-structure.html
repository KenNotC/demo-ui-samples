Site Structure

How GlobalDocs Atlas organises pages and navigation — and how to extend it for your own documentation.

How pages work

Each page is a self-contained HTML file. There is no server, no build process, and no shared state between pages. The full template explains the three structural sections every page shares and which parts to edit versus which to leave unchanged.

Navigation groups

The sidebar organises links into four groups. The full template includes complete instructions for renaming, reordering, and adding groups, as well as a naming convention table for new pages.
